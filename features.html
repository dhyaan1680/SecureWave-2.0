Features

mmWave Detection

Micro-motion detection using Doppler-based mmWave sensing. Works through paper, wood, concrete, glass.

Tamper & Anti-jam

Built-in tamper sensors, frequency hopping and proprietary channels reduce jamming risk.

Fingerprint Acknowledge

Local fingerprint sensor on the base unit required to silence and acknowledge alarms.

On-prem Software Suite

Local hosting, no internet required — integrates with NVR and emergency systems.

Battery Backup

Sensors run on internal backup battery (1 month) and base unit keeps memory of logs.

Scalable

Support for up to 128 sensors with unique IDs and configurable floorplan mapping.
